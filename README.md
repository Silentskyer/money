# money

一個吉普力風格的記帳網站，支援收入/支出、時間紀錄、剩餘可用金額顯示，並可依月份與年份篩選與索引。

## 功能
- 新增收入/支出、品項與時間
- 自動計算整體與期間剩餘可用金額
- 依年份/月份篩選
- 依時間或品項排序與搜尋
- 本地 `localStorage` 保存

## 使用方式
1. 直接開啟 `index.html`
2. 輸入品項、金額、類型與時間後加入清單
3. 使用篩選與索引查看不同月份/年份的剩餘金額

## 檔案結構
- `index.html`：頁面結構
- `styles.css`：吉普力風樣式
- `app.js`：記帳邏輯與資料儲存
- `WORKPLAN.md`：工作計畫
