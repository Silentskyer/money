# 記帳網站開發工作計畫

最後更新：2026-03-16

## 目標
建立一個吉普力風格的記帳網站，支援收入/支出輸入、時間紀錄、剩餘可用金額顯示、依月份與年份統計，以及時間或品項索引。

## 工作SOP（勾選清單）
- [ ] 需求確認：整理欄位與功能範圍（收入/支出、時間、品項、月份/年份統計、索引方式）
- [ ] 資料模型設計：規劃記帳資料結構與統計邏輯
- [ ] 介面框架：首頁、輸入表單、列表、統計視圖、篩選/索引區塊
- [ ] 吉普力風格視覺：色彩、字體、背景與插畫/紋理方向
- [ ] 功能實作：新增/刪除/編輯記帳項目與剩餘金額計算
- [ ] 月年統計：依月份與年份的剩餘可用金額彙整
- [ ] 索引與篩選：依時間與品項查詢
- [ ] 測試與修正：基本流程與邏輯驗證
- [ ] 文件與交付：使用說明與後續擴充建議

## 里程碑
- 里程碑 1：需求與資料模型完成
- 里程碑 2：核心功能可用
- 里程碑 3：視覺風格完成
- 里程碑 4：索引與統計完成
